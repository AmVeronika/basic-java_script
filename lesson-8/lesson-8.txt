1) Возможно при назначении ячеек с едой и барьером можно испольщовать одну функцию...
Данная тема еще несовсем понятна мне в рамках данной задачи. 
Только на более низком уровне - надо создать переменную, недоступную в глобальной области видимости, за исключением переменных, которые ссылаются на данную 
функцию с передаей нужного параметра...

a)  
if (!("a" in window)) {
    var a = 1;
}
alert(a);
Ответ: Глобальный объект window видит, что в коде создана переменная "a". Но назначена она локально. 
В условии указано, если в window нет "а", то присвоить ей значение 1.
Так как а есть глобально, но значение не присвоено, выведется undefined.

b)
var b = function a(x) {
    x && a(--x);
};
alert(a);
Ответ: Переменная "а" не создана. Выведет ошибку

с)
function a(x) {

d)
function b(x, y, a) {
    arguments[2] = 10;
    alert(a);
}
b(1, 2, 3);

Ответ: Третий элемент с индексом [2] в парметре функции b  - это а.
При вызове функции b(1,2,3) - запись  arguments[2] = 10 перезаписывает параметр а на 10. В итоге, ответ: 1,2, 10.
